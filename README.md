# Proyecto final IMT 2200 : Distribución y disponibilidad de centros de salud públicos y farmacias en Chile

Link repositorio : [github repo](https://github.com/Utmite/imt2200-project.git)

## a) Objetivos 
1. Analizar la distribución de servicios médicos que se encuentran a lo largo
del territorio chileno.
2. Visualizar las localidades donde se pueden maximizar los beneficios a la
población según la distribución.


## b) Fuente de Datos : Públicos y de libre acceso
1. Instituto de Salud Pública (ISP)
2. Ministerio de Salud (MINSAL)
3. Instituto Nacional de Estadísticas (INE) : Censo 2017
4. Biblioteca del Congreso Nacional
